Développement

Comment ce projet a été fait?

Pour faire ce projet nous avons utilisé tous les outils disponibles pour rendre ce travail le plus semblable du vrai site play2048.co. Nous avons employé Visual Studio Code comme outil de développement dans le but de rendre ce tp le plus proche de notions appris dans les séance de cours et de démonstrations. Pour partager le projet nous avons établi un repertoire sur GitHub. On a déployé également des versions standalone des libreries «reactjs» et «babel» pour permettre un deployment effectif sur le seveur du diro.
